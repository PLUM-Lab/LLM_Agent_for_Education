# Medical Case Study Quiz System

A web-based medical case study quiz system with integrated ChatGPT for interactive learning.

## Quick Start

### 1. Configure API Key

**IMPORTANT**: You must configure your OpenAI API key before using the chat feature.

1. Open the file `api-key.js` in your editor
2. Find this line:
   ```javascript
   const OPENAI_API_KEY = 'sk-your-api-key-here';
   ```
3. Replace `'sk-your-api-key-here'` with your actual OpenAI API key
4. Save the file (Ctrl+S)

**Get your API key from:** https://platform.openai.com/account/api-keys

**Note**: The `api-key.js` file is in `.gitignore` and will NOT be committed to Git. Your API key stays local and secure.

### 2. Start the UI



*
1. Double-click `open-medical-quiz-simple.bat` file
2. The UI will open in your browser
