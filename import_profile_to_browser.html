📥 导入学生档案到浏览器

Import Student Profile to Browser

使用说明 / Instructions:
  1. 运行测试脚本生成学生档案JSON文件： python test_knowledge_tracking.py --student-name "TestStudent" --report "report.txt"
  2. 选择生成的 student_profile_*.json 文件
  3. 点击"导入到localStorage"按钮
  4. 打开 medical-quiz.html，输入相同的学生姓名并初始化档案
  5. 你将看到完成所有问题后的知识追踪结果！
