🚀 Quick Start Guide

Method 1: Use VS Code Live Server (Easiest)

If you're using VS Code:

  1. Install the "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"

Method 2: Use Python (if installed)

Open PowerShell or Command Prompt in this folder and run:

python -m http.server 8000

Then open: http://localhost:8000

Method 3: Use Node.js (if installed)

Open PowerShell or Command Prompt in this folder and run:

npx http-server -p 8000

Then open: http://localhost:8000

Method 4: Double-click to open (Limited)

You can directly open index.html in your browser, but JSON loading may not work due to CORS restrictions.
